# 开关

## 说明

开关允许用户开启或关闭一个选项。

## 示例

```json
{
    "id": "switch",
    "name": "开关",
    "type": "switch"
}
```

## 默认属性

属性 | 说明 | 类型 | 默认值
:-- | :-- | :-- | :--
description | 描述 | String | ''
dist | 输出值 | Object | {value: false}
dynamic | 发布后是否可修改 | Boolean | true
name | 名称 | String | ''
type | 类型 | String | 'switch'
vip | 是否为付费功能 | Boolean | false
