# 颜色

## 说明

颜色允许用户从色板中选取颜色。

## 示例

```json
{
    "id": "barColor",
    "name": "进度条颜色",
    "type": "color"
}
```

## 默认属性

属性 | 说明 | 类型 | 默认值
:-- | :-- | :-- | :--
description | 描述 | String | ''
dist | 输出值 | Object | {value: 'transparent'}
dynamic | 发布后是否可修改 | Boolean | true
name | 名称 | String | ''
type | 类型 | String | 'color'
vip | 是否为付费功能 | Boolean | false
