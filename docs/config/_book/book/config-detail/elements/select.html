下拉框

下拉框

说明

下拉框允许用户选择一个选项。

示例

{
    "level": {
        "name": "游戏难度",
        "options": [
            {
                "value": "easy",
                "text": "简单"
            },
            {
                "value": "normal",
                "text": "普通"
            },
            {
                "value": "hard",
                "text": "困难"
            }
        ],
        "type": "select",
        "dist": {
            "value": "easy"
        }
    }
}
